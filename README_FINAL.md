# Anti-Attack AI

**Protect your company from cyber attacks in real-time with automated detection and response.**

---

## Overview

Anti-Attack AI is a complete endpoint security system designed to:

- Monitor system activity (CPU, RAM, processes)
- Detect abnormal behavior and potential cyber threats
- Activate Safe Mode automatically to prevent damage
- Send immediate email alerts for any suspicious activity
- Log all events for auditing and review

---

## Features

1. **Real-time Monitoring** – Detect threats as they happen  
2. **Automated Response** – Safe Mode triggers automatically  
3. **Instant Alerts** – Email notifications to designated addresses  
4. **Flexible Configuration** – Easy adjustments via `config.sh`  
5. **Easy Installation** – One-command installer (`install.sh`)  
6. **Auto-start Service** – Runs automatically on system startup  
7. **Privacy-focused** – No sensitive company data is accessed
