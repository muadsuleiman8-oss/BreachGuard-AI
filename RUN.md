# How to Run Anti-Attack AI

This guide explains how to start and stop the Anti-Attack AI system safely.

## Files
- `monitor.sh` – Monitors system behavior
- `safe_mode.sh` – Activates Safe Mode in case of suspicious activity
- `alerts.log` – Logs all warnings and Safe Mode activations

## Steps to Run

1. Make scripts executable:
```bash
chmod +x monitor.sh safe_mode.sh
